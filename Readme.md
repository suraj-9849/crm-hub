# CRM Hub

A unified platform integrating API routes for both Hubspot and Zoho CRMs.

## Features
 - OAuth Flow
  - Meetings, Tasks, Notes, Emails, and Deals APIs

## Tech Stack

- **Backend:** NestJS
- **Frontend:** React

## Installation

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/suraj-9849/crm-hub.git
   cd crm-hub
   ```

2. **Install Dependencies:**

### Go to the Specific CRM it can be Hubspot or Zoho then Run this commands:

   - For the backend:
     ```bash
     cd backend
     npm install
     ```
   - For the frontend:
     ```bash
     cd ../frontend
     npm install
     ```

3. **Run the Application:**
   - Start the backend:
     ```bash
     npm run start:dev
     ```
   - Start the frontend:
     ```bash
     npm run dev
     ```
